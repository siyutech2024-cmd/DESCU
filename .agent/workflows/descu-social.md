## 发布工作流

### 手动发布清单
1. [ ] Instagram - 发布帖子 + Story
2. [ ] Facebook - 发布帖子 + 分享到相关群组
3. [ ] X/Twitter - 发布推文/线程
4. [ ] Reddit - 发布到相关社区（注意不要spamming）

### 自动化工具推荐
| 工具 | 平台 | 功能 |
|------|------|------|
| Buffer | 全平台 | 定时发布 |
| Later | Instagram | 可视化排程 |
| Typefully | X/Twitter | 线程发布 |
| Publer | 全平台 | 免费多账号 |

### 发布频率建议
- Instagram: 1帖子/天 + 3-5 Stories
- Facebook: 1帖子/天
- X/Twitter: 3-5 推文/天
- Reddit: 1-2 帖子/周（避免被标记spam）
